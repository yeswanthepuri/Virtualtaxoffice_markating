Full-featured software at a fair price

The way in which we can offer this full-featured software to small and big businesses alike is through infrastructure usage based on users and disk storage needed, which does affect our cost of providing the system for subscribers. We do not charge for bandwidth usage, or for any activity on a per-record basis, so you can use the system as often as you like without worry.

However, if certain subscribers are very heavy users, we do need to increase our server capacity and internet bandwidth, which affects our cost that we need to pass on to those higher volume users. In order to do this equitably and transparently, our add-on pricing for users and storage per module is summarized above and more fully explained below:
